Project

GT Movies Store — Web App

GT Movies Store is a premier online destination for a cinematic adventure like no other! Dive into the world of movies from the comfort of your own home with our user-friendly and immersive online movie store.

Screens & Features aligned to User Stories

  • About GT Movies Store Information about GT Movies Store is present on the landing page and in the About tab
  • register account Register account under the SignUp tab in the header
  • log in Log in under the Login tab in the header
  • view the list of movies View the list of movies under the movies tab in the header
  • search movies per title Search movies by going to the movies page and clicking the search bar
  • access a shopping cart Access the shopping cart by clicking the cart tab in the header
  • add one or more items of a movie to my shopping cart Add one or more items of a movie to my shopping cart by going to the detail page of a movie and clicking the add to cart button
  • create movie reviews Create movie reviews by going to the detail page of a movie and clicking the add review button
  • remove all movie items from my shopping cart Remove all movie items from my shopping cart by going to the cart page and clicking the remove all movies from Cart button
  • edit reviews Edit your reviews by going to the detail page of a movie and clicking the edit review button
  • delete my movie reviews Delete your reviews by going to the detail page of a movie and clicking the delete review button
  • see the reviews of a movie See the reviews of a movie by going to the detail page of a movie by clicking on the movie title in the movies Pages
  • see the details of a movie See the details of a movie by going to the detail page of a movie by clicking on the movie title in the movies Pages
  • see a list of your orders See a list of your orders by going to the Orders tab in the header
  • accessibility features The GT Movies Store can be accessed from any desktop using a web browser, so it can be used anywhere with an internet connection. The GT Movies Store features a responsive Graphical User Interface (GUI) so it can be accessed through different devices with diverse screen sizes.
  • admin features Admin features can be accessed by logging in as an admin user and going to the navigating to the `/admin` endpoint. There is no button to access here for security reasons. You would be able to manage (view, create, update, or delete) users, movies, reviews, and orders from the GT Movies Store so we can keep information up to date.
